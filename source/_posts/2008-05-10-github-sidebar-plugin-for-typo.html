--- layout: post title: GitHub Sidebar Plugin for Typo tags: - Blogging - Rails status: publish type: post published: true meta: _edit_last: '2' --- I liked Dr Nic's github badge so much that I wrapped a sidebar plugin around it for Typo users. You can get it from github. Or install it into typo like this: 
# cd typo/vendor/plugins 
# git clone git://github.com/peteonrails/githubsidebar.git github_sidebar
